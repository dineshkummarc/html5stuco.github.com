But why would I put something there?

  • Store user preferences
  • Save the state of the application
  • Cache scripts and assets locally (up to 5mb!)

History

  • Cookies
  • Flash LSO (local shared object)
  • Silverlight Isolated Storage, Google Gears

Setup!

if('localStorage' in window && window['localStorage'] !== null){
  var store = window.localStorage;
  /*
   * Storage things here!
   */
}

Save!

store.setItem('key','value');
store.setItem('carnegie', 'mellon');

store.getItem('key'); // Returns 'value'
store.getItem('carnegie'); // Returns 'mellon'

localStorage is an object...

store['molly'] = 'samuels';
store.brian = 'yee';

store['molly']; // Returns 'samuels'
store.brian; // Returns 'yee'

...but you can't store them

var foods = {
  'fruit':'apple',
  'meat':'steak'
}
store['noms'] = foods;
store['noms']; // returns [object Object] :(

store['noms'] = JSON.stringify(foods);
store['noms']; // Returns our foot object!
